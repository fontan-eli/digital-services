# Concierge IT Services

This project is a simple responsive website for a Concierge IT Services company. It is built using HTML, CSS, and vanilla JavaScript, without any frameworks.

## Project Structure

```
concierge-it-services
├── index.html
├── styles.css
├── script.js
└── README.md
```

## Files Description

- **index.html**: Contains the structure of the website, including a navigation bar with links to different sections (Home, Who We Are, Scope of Services, Contact Us, Testimonials) and the corresponding content sections. Each link scrolls to its respective section on the same page.

- **styles.css**: Contains the styling for the website, including styles for the navigation bar, sections, and responsive design elements to ensure the website looks professional and modern on both desktop and mobile devices.

- **script.js**: Contains JavaScript functionality for the website, including smooth scrolling when navigation links are clicked.

## How to Run the Website

To run the website locally, follow these steps:

1. Download or clone the repository to your local machine.
2. Navigate to the project directory.
3. Open `index.html` in your web browser.

The website should display correctly, and you can navigate through the sections using the navigation bar.
